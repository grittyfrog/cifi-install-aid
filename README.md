# CiFi Install Aid

A visual reference tool for Cell: Idle Factory Incremental (CiFi)'s spaceship installation grid. Shows all ship hex layouts with boost types, generator indicators, and cell output ratios.

Click any hex to see what it does. Use the ◂ ×N ▸ stepper on each card to multiply cell ratios and compare equivalent point allocations across nodes.

**[Live version](https://grittyfrog.github.io/cifi-install-aid/)**

Built with [uhtml](https://github.com/WebReflection/uhtml) and Vite. Cards are pure SVG with HTML controls and tooltips. The build produces a single self-contained HTML file that works without JavaScript via build-time prerendering.
